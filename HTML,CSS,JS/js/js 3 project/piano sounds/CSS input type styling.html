form submit krne pr page refresh bnd kese krte hai
ChatGPT said:

Aap form submit hone par page refresh ko rok sakte hain JavaScript se. Iske liye event.preventDefault() use hota hai.

1. Simple JavaScript Method

html
<form id="myForm">
  <input type="text" placeholder="Name">
  <input type="submit" value="Submit">
</form>

<script>
  document.getElementById("myForm").addEventListener("submit", function(e) {
    e.preventDefault();   //  Page refresh ko rok deta hai
    console.log("Form Submitted Without Refresh!");
  });
</script>

2. Inline JavaScript Method

(Short method)

html
<form onsubmit="event.preventDefault()">
  <input type="text">
  <input type="submit">
</form>

3. Agar aap button ke click par handle karna chahte ho

html
<form id="form">
  <input type="text">
  <button id="btn">Submit</button>
</form>

<script>
document.getElementById("btn").addEventListener("click", function(e){
  e.preventDefault();
  console.log("No refresh!");
});
</script>

IMPORTANT

  • Form default action = page refresh

  • event.preventDefault() = refresh stop
